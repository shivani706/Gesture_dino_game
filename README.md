# 🎮 Gesture Dino Game

A gesture-controlled version of the classic Chrome Dino Game built using **Python**, **OpenCV**, **MediaPipe**, and **Pygame**. Control the dinosaur using your hand gestures through your webcam—no keyboard required!

## ✨ Features

* 🖐️ Real-time hand gesture recognition
* 🎥 Webcam-based controls
* 🦖 Dino jumps and ducks using gestures
* 🌵 Random obstacle generation
* 🎵 Background music and sound effects
* 📈 Score tracking
* 🖥️ Windows executable included

## 🛠️ Technologies Used

* Python 3.11
* OpenCV
* MediaPipe
* Pygame
* NumPy

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/shivani706/Gesture_dino_game.git
cd Gesture_dino_game
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the game:

```bash
python main.py
```

## 🎮 Controls

* ✋ Open Hand → Jump
* ✊ Closed Fist → Duck
* Keep your hand clearly visible in front of the webcam.
